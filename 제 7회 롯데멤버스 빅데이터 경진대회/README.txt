# 제 7회 롯데멤버스 빅데이터 경진대회

<br/>

## 1. 배경 & 목적
 
- 배경 : 고객 구매 데이터에 기반한 예측 모델 개발 및 개인화 마케팅 전략 제안
- 주제 : 고객 별 유형 분석 및 개인화 넛지 마케팅

<br/>

## 2. 주최/주관 & 성과

- 주최/주관: 롯데멤버스 
- 운영: AI Factory
<br/>

<br/>

## 3. 프로젝트 기간

- 대회 기간 : 33.06.02 ~ 07.03

<br/>

## 4. 프로젝트 소개

&nbsp;&nbsp;&nbsp;&nbsp; 내부에서 제공하는 고객 경험 데이터와 L.Pay 사용 데이터를 PDDE(유통사)와 COP_U(제휴사)의 관점 등 제공 되는 데이터로부터 여러 관점을 중심으로 분석을 진행. 이를 바탕으로 다양한 Feature를 생성함. AutoML과 TabNEt을 중심으로 예측 모델을 생성. 해당 모델을 바탕으로 다양한 클러스터링 모델 적용. PCA, T-SNE와 같은 여러 시각화 모델을 통해 시각화. 해당 클러스터링을 바탕으로 개인화 넛지 마케팅을 제안함
<br/>
